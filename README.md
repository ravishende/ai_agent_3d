# ai_agent_3d

An agent that uses policy iteration to run through a 3d Temple Run / Subway Surfers style environment

# Setup

1. Create and run virtual environment
   `python3 -m venv .env`
   `source .env/bin/activate`
2. Install required packages
   `python3 -m pip install -r requirements.txt`

# Run

Project is still in progress.

- To see the environment, run `python3 env_3d.py`

- To be able to play through a text based environment, run `python3 play.py`
